"Hallo Welt" oder eine blinkende LED

Das erste Programm (in der Arduinowelt Sketch genannt) lässt eine LED, die mit einem Vorwiderstand verbunden ist, regelmäßig blinken. Am folgenden Programmbeispiel wird der Grundaufbau eines Arduino-Sketches und alle notwendigen Befehle erläutert. Sollten jedoch Fragen zu einzelnen Befehlen auftauchen oder einfach nur Interesse an mehr Informationen aufkommen, ist die Arduino-Referenz die beste Anlaufstelle.

Einfache Schaltung für eine LED

Bevor jedoch der Arduino programmiert werden kann, muss die LED mit einer entsprechenden Schaltung angeschlossen werden. Wichtig ist bei der LED, dass der lange Anschluss der Plus-Pol und der kurze Anschluss der Minus-Pol ist. Wird die LED falsch angeschlossen, wird sie nicht leuchten. Außerdem muss dafür gesorgt werden, dass der Strom durch die LED auf ca. 20 mA begrenzt wird. Ein größerer Strom könnte die LED zerstören. Auch sollten die Ströme der einzelnen Arduino-Pins auf 20 mA begrenzt werden. Höhere Stromwerte könnten den Arduino zerstören. Zur Strombegrenzung wird ein Widerstand mit \( R=220\ \Omega \) verwendet. Die vollständige Schaltung ist unten abgebildet.

Eine LED sollte niemals ohne Vorwiderstand an den Arduino angeschlossen werden! Dies könnte den Arduino und die LED zerstören.

Bestimmung des Vorwiderstandes

Warum wird aber ein Widerstand von \( R=220 \Omega \) verwendet?

Betrachtet man das Schaltbild oben rechts, so lässt sich folgende Masche aufstellen \( U_\text{Pin} = U_R + U_D\). Da für \(U_R = I \cdot R\) gilt, lässt sich die Gleichung auch folgendermaßen schreiben $$ U_\text{Pin} = I\cdot R + U_D$$ und nach dem gesuchten Widerstand \(R\) umstellen: $$ R= \frac{U_\text{Pin}- U_D}{I} $$ Mit den gegebenen Werten für die LED \( U_D = 2\ \text{V} \) und \( I=20\ \text{mA}\) folgt: $$ R= \frac{U_\text{Pin} - U_D}{I}= \frac{5\ V - 2\ V}{20\ mA} = 150\ \Omega < 220\ \Omega$$

Sollten in einer Schaltung mehrere LEDs verwendet werden, muss jede LED mit einem eigenen Widerstand verbunden werden.

Aufgaben

  1. Bestimmen Sie für eine an 5V angeschlossene blaue LED den benötigten Vorwiderstand, damit ein Maximalstrom von 20 mA bei einer Spannung von 3,2 V nicht überschritten wird.

Blink-Programm

Ein Arduino-Programm besteht immer aus zwei Teilen, welche durch die beiden Funktionen void setup() und void loop() dargestellt werden.

  • Die Funktion void setup() wird vom Arduino immer genau einmal ausgeführt und eignet sich damit besonders gut zum Konfigurieren des Boards. Hier kann zum Beispiel angegeben werden, ob ein Pin als Ein- oder Ausgang genutzt werden soll, oder ob eine serielle Kommunikation zum PC aufgebaut werden soll.
  • Die Fuktion void loop() wird, nachdem setup() ausgeführt wurde, wie eine Endlosschleife immer wieder aufgerufen. In aller Regel befindet sich hier das Hauptprogramm. Für das Beispiel der blinkenden LED wird hier immer wiederkehrend die LED erst an- und dann ausgeschaltet.
/* Grundaufbau eines Arduino-Sketches (Programms)*/

void setup() 
{
  /* Die setup() Funktion wird nur einmal durchlaufen und konfiguriert das Board.
   * Hier werden z.B. Pins als Ein- oder Ausgänge definiert.
   */
}

void loop() 
{
  /* Die loop() Funktion wird immer wieder ausgeführt und stellt eine Endlosschleife
   * dar. In diesem Teil des Arduino-Programms befindet sich das Hauptprogramm.
   */
}

Sollte eine der beiden Funktion setup() oder loop() für ein Programm nicht benötigt werden, so wird dessen Rumpf einfach leer gelassen. Sie dürfen niemals ganz weggelassen werden, da sonst Fehlermeldungen beim Kompilieren auftreten.

Im folgenden Programmbeispiel wird eine LED mit einer Periodendauer von 2 Sekunden an- und ausgeschaltet. Die Erläuterungen zu den einzelnen Befehlen finden Sie im Anschluss.

Hallo_Welt.ino
/* Die angeschlossene LED wird abwechselnd an- und ausgeschaltet.
Durch den Befehl delay() ergibt sich eine blinkende LED.*/

#define PIN_LED 9   // im Programm wird PIN_LED mmer durch 9 ersetzt
                    // #define definiert einen Platzhalter, keine Variable

// setup() wird nur einmal durchlaufen und konfiguriert das Board
void setup() {
  pinMode(PIN_LED,OUTPUT); // Setzt den Pin (9) der roten LED als Ausgang
}

// loop() läuft als Endlosschleife. Hier kommt das Hauptprogramm rein. 
void loop() {
  digitalWrite(PIN_LED,HIGH);  // Ausgang HIGH (5V), LED an
  delay(1000);                 // Programmverzögerung um 1s = 1000ms
  digitalWrite(PIN_LED,LOW);   // Ausgang LOW (0V), LED aus
  delay(1000);                 // 
}

Erläuterungen zum Programm

  • #define PIN_LED 9 definiert eine Konstante (Platzhalter), die nicht im Speicher abgelegt wird, sondern beim Kompilieren im Programmcode mit dessen Wert ersetzt wird. Die Zeile endet nicht mit einem Semikolon.
  • void setup(){...} ist eine Funktion, welche beim Starten des Arduino-Sketches genau einmal ausgeführt wird.
  • Die Funktion void loop(){...} wird in einer Endlosschleife ausgeführt.
  • Zum Konfigurieren der Pins als Eingang bzw. Ausgang wird der Befehl pinMode(pin,INPUT) bzw. pinMode(pin,OUTPUT) verwendet.
  • Wenn ein Pin als Ausgang definiert ist, kann mit dem Befehl digitalWrite(pin,HIGH) der entsprechende pin auf HIGH, also 5 V, gesetzt werden. Mit digitalWrite(pin,LOW) wird der Pin auf 0 V gesetzt.
  • Der Befehl delay(time_ms) pausiert den Programmablauf um den angegebenen Wert in Millisekunden.

Aufgaben

  1. Ändern Sie die Periodendauer der LED auf 2 Sekunden. Innerhalb der 2 Sekunden soll die LED 90% der Zeit ausgeschaltet sein.
  2. Lassen Sie die LED das SOS Signal morsen (3x kurz, 3x lang, 3x kurz).
  3. Schließen Sie eine weitere LED an einen anderen Pin an und lassen Sie beide LEDs abwechselnd blinken.
  4. Lassen Sie die zwei LEDs mit unterschiedlichen An- und Auszeiten blinken.
